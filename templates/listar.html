Programação Funcional em Python

Funções Puras

Funções que não alteram estado externo, não têm efeitos colaterais e retornam sempre o mesmo resultado para as mesmas entradas, independente do contexto de execução.

Maiúsculas (para_maiusculo)

    {% for nome in nomes_maiusculos %}
  • {{ nome }}
    • {% endfor %}
Função Pura Python
# Função pura que converte texto para maiúsculo def transformar_maiusculo(texto: str) -> str: return texto.upper()

Map

Função de alta ordem que transforma cada elemento de uma coleção aplicando uma função, sem modificar a coleção original e criando uma nova coleção com os resultados.

Primeira letra dos nome em maiusculo(lambda)

    {% for nome in nomes_em_titulo %}
  • {{ nome }}
    • {% endfor %}
Operação Map Python
# Exemplo com map e lambda nomes_em_titulo = list(map(lambda nome: nome.title(), nomes)) # Exemplo com função nomeada def transformar_titulo(texto: str) -> str: return texto.title() nomes_em_titulo = list(map(transformar_titulo, nomes))

Filter

Função de alta ordem que cria uma nova coleção contendo apenas os elementos que atendem a uma condição específica (predicado), permitindo seleção declarativa de dados.

Nomes com mais de 5 caracteres

    {% for usuario in nomes_longos %}
  • {{ usuario.nome }} ({{ usuario.email }})
    • {% endfor %}
Operação Filter Python
# Filtrando usuários com nomes longos nomes_longos = list(filter(lambda usuario: len(usuario.nome) > 5, usuarios))

Reduce

Função de alta ordem que itera sobre uma coleção, acumulando resultados segundo uma função, transformando múltiplos valores em um único valor agregado (fold operation).

Total de caracteres em todos os nomes

{{ total_caracteres }} caracteres
Operação Reduce Python
# Somando comprimentos dos nomes total_caracteres = reduce(lambda acc, nome: acc + len(nome), nomes, 0)

Composição de Funções

Técnica que combina duas ou mais funções para criar uma nova função: f(g(x)). O resultado de uma função torna-se a entrada da próxima, permitindo criar operações complexas a partir de funções simples e reutilizáveis.

Minúsculo → Capitalizado

    {% for nome in nomes_processados %}
  • {{ nome }}
    • {% endfor %}
Composição de Funções Python
# Função de alta ordem para composição de funções (f ∘ g) def compor_funcoes(f, g): return lambda x: f(g(x)) # Composição de funções titulo_apos_minusculo = compor_funcoes(transformar_titulo, transformar_minusculo) nomes_processados = [titulo_apos_minusculo(nome) for nome in nomes]

List Comprehension

Sintaxe concisa para criar novas listas aplicando expressões a cada item de uma coleção, opcionalmente filtrado por condições. Combina map e filter em uma construção elegante e legível, específica de Python.

Iniciais dos Nomes

{% for inicial in iniciais_nomes %} {{ inicial }} {% endfor %}
List Comprehension Python
# Extrai a primeira letra de cada nome def extrair_iniciais(nomes): return [nome[0].upper() for nome in nomes if nome] # Extraindo iniciais dos nomes iniciais_nomes = extrair_iniciais(nomes)
